# IEFI_CASINO_DADOS

La lógica de juego es la misma con la diferencia que ahora se puede apreciar visualmente. 
Para jugar correctamente como primer paso es cargar el nombre y el apodo de los jugadores, luego seleccionar el tipo de jugador.
Tiene otra opción mucho más rápida (Botón cargar partida) que permite cargar una partida automáticamente y el juego comienza enseguida.
Si se sigue con la carga manual, luego de la carga de los jugadores debe configurar los datos de la partida, dinero inicial, cantidad de rondas, cantidad de partidas y confirmar por último la partida, una vez confirmada el juego estará listo para comenzar.

Al iniciar el juego, en el centro de la ventana hay un botón que permite avanzar ronda a ronda. En cada rondase visualiza el ganador y si algún jugador fue confundido por la trampa. Al finalizar el juego, el sistema genera un reporte con el ranking de los jugadores, estadísticas del juego y el historial de las partidas.

El menú interactivo tiene diversas funcionalidades:
* Partida: incluye las opciones Pausa, Guardar y Salir.
* Ver: permite acceder al Ranking de jugadores, al Historial de partidas y a las Estadísticas Generales, que muestran la mayor apuesta, el mejor puntaje con los dados y las veces que el casino utilizó dados cargados.

El juego tiene incluido base de datos SQLite, por lo que permitirá un manejo de datos mucho más dinámico, dejando de esta manera casi por obsoleto el uso de los archivos de texto que se manejaba con anterioridad para la persistencia de los datos.

A lo largo del proyecto se fueron presentando muchos desafíos, ya que el código era más extenso y poder seguir el mismo flujo y estructura para que no pierda el hilo el programa se complicaba, el desafío estuvo bueno y sirvió para aprender acerca de todo lo que tiene Java para ofrecer.
