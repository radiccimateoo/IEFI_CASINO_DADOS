# IEFI_CASINO_DADOS

La logica de juego es la misma con la diferencia que ahora se puede apreciar visualmente. 
Para jugar correctamente como primer paso es cargar el nombre y el apodo de los jugadores, luego seleccionar el tipo de jugador
Tiene otra opción mucho mas rápida (Boton cargar partida) que permite cargar una partida automaticamente y el juego comienza enseguida.
Si se sigue con la carga manual, luego de la carga de los jugadores debe configurar los datos de la partida, dinero inicial, cantidad de rondas
cantidad de partidas y confirmar por ultimo la partida, una vez confirmada el juego estara listo para comenzar.

Cuando se inicia el juego, en el medio de la ventana hay un boton que permite ir jugando cada una de las rondas en donde se va mostrando el ganador
de cada una, si algun jugador fue confundido por la trampa, entre otras cosas, finalizado el juego muestra un reporte con el ranking de los jugadores, estadísticas del juego y el historial de las partidas.

Tambien ofrece un panel extra llamado "Ver" que permite ver historial de partidas, partidas guardadas y estadisticas generales

El juego tiene incluido base de datos SQLite, por lo que permitira un manejo de datos mucho mas dinamico, dejando de esta manera casi por obsoleto el uso
de los archivos de texto que se manejaba con anterioridad para la persistencia de los datos.

A lo largo del proyeto se fueron presentando muchos desafios, ya que el codigo era mas extenso y poder seguir el mismo flujo y estructura para que no pierda
el hilo el programa se complicaba, el desafio estuvo bueno y sirvio para aprender a cerca de todo lo que tiene Java para ofrecer
